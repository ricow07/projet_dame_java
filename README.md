# Projet de Programmation : Jeu de Dames en Java

**Membres du groupe : Romain Tholle et Anguelo Carathanasis   

## Présentation du projet
Ce projet est une implémentation classique du jeu de dames international (plateau 10x10) développée entièrement en Java. Il permet à deux joueurs de s'affronter sur le même ordinateur, en respectant les règles officielles : déplacements en diagonales, prises obligatoires, rafles (prises multiples) et promotion en dames.
